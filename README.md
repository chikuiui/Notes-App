# NoteTakingApp 

This App is used to store,update the notes having Title and Description in Room Database and showing in the form of Staggered Manner.


## Key Components of this App
* MVVM Architecture
* View Model
* Room Database
* Navigation Components
* SafeArgs
* SearchView
* DataBinding
* CardView
* RecyclerView
